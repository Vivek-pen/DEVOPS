# Docker

Install docker desktop

check installation with  docker --version and docker info

Image - blueprint of your program

container - running instance of the image

Volume - storing data, DockerFile - used to store instructions to build an image

pull image - docker pull nginx:tag

docker --version
1) docker pull nginx
2) docker run -d --name testweb -p 8080:80 nginx      //running a docker container

3) docker images - display pulled images
4) docker ps - display running containers
5) docker ps -a - display all containers

7) docker start <cont_id> - start a stopped container
6) docker stop <cont_id> - stop running containers
7) docker rm <cont_id> - remove containers
8) docker rmi nginx - remove image
